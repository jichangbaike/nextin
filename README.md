**以下是针对Nextin客户端的核心简介**：

## 核心技术架构

内核驱动：采用 Clash Meta (Mihomo) 内核，具备极强的协议兼容性和分流规则处理能力。

原生开发：基于 SwiftUI 构建，界面简洁、交互流畅，完美适配 Apple 各平台的系统特性。


## Nextin 支持平台

Nextin 实现了 Apple 全平台覆盖：

iOS / iPadOS：支持 iOS 15.0 及以上版本。

macOS：支持 macOS 12.0 及以上（需 M1 芯片及后续版本）。

Apple TV：支持 Apple TV 17.0 及以上版本。

Apple Vision Pro：原生适配 visionOS。


## Nextin 协议支持
支持目前主流及前沿的所有代理协议：

常规协议：Shadowsocks (SS/SSR)、VMess、Trojan。

新一代协议：VLESS (Reality / XTLS / XHTTP)、Hysteria / Hysteria2、TUIC。

传输层：支持 WebSocket、gRPC、HTTP/2 等。


## Nextin 关键功能特性

- 快速导入：兼容 Clash 和 Shadowrocket 格式的订阅链接；支持通过服务商标识符快速配置。

- 智能分流：内置完善的流量路由规则，支持自定义路由逻辑和订阅分流编辑。

- 极致体验：支持实时网速显示、控制中心快捷开关（Toggle）、桌面小组件（Widgets）。

- 离线操作：支持在不启动代理的状态下切换节点和分流模式。

- 内置客服：整合了服务商客服入口，用户可直接在 App 内联系技术支持。


## 适用场景与评价

定位：介于 Shadowrocket（小火箭）的极简与 Stash/Quantumult X 的极繁之间。

优点：配置简单（对新手友好），支持 Apple TV 和 Vision Pro（覆盖面广），内核更新及时。

缺点：目前主要深耕 Apple 生态，暂无 Android 或 Windows 版本。
